# 🎨 Промпт для GPT-5 Pro: Изменение цветовой схемы виджета РТС

## 📋 ЗАДАНИЕ

**Контекст:** Виджет автоматизации для работы с государственными/коммерческими закупками на платформе РТС-тендер

**Цель:** Создать новую профессиональную цветовую схему, подходящую для делового B2B-инструмента

---

## 🎯 ТРЕБОВАНИЯ К ДИЗАЙНУ

### ✅ Обязательно:
- **Профессиональный деловой стиль** - виджет для работы с закупками и тендерами
- **Премиальный внешний вид** - надежность и качество инструмента
- **Спокойные, не агрессивные цвета** - длительная работа без утомления глаз
- **Отличная читаемость** - четкий контраст текста и фона
- **Гармония с белым фоном сайта** - виджет должен естественно вписываться
- **Сохранить эффекты glassmorphism** - полупрозрачность и размытие

### ❌ Ограничения:
- **НЕ использовать черный/красный** - как на основном сайте РТС
- **НЕ использовать яркие кричащие цвета** - не должен резко выделяться
- **НЕ использовать теплые "дружелюбные" тона** - персиковый, коралловый не подходят для B2B
- **НЕ менять структуру CSS** - только цвета и градиенты

## 🔧 ЧТО НУЖНО ИЗМЕНИТЬ

### Изменить все цвета в:
- `color` - цвета текста
- `background` - фоны элементов
- `border-color` - цвета рамок
- `linear-gradient()` - все градиенты
- `radial-gradient()` - радиальные градиенты
- `box-shadow` - цвета теней (включая rgba)
- Все состояния: `:hover`, `:focus`, `:active`, `[data-state="..."]`
- Цвета индикаторов статусов (success, warning, error)

### НЕ изменять:
- Структуру CSS классов
- Значения `padding`, `margin`, `border-radius`
- Анимации `@keyframes` и `transition`
- Размеры элементов
- Z-index и позиционирование
